# yourConnector 安装与环境准备 v0

## 1. 安装目标

- Rust relay / sidecar 开发
- Tauri iOS 调试
- 协议层（Rust + Schema）联调

## 2. 推荐安装顺序

1. `Xcode Command Line Tools`、`Homebrew`、`Git`
2. `rustup`、`cargo`
3. `Xcode`、`CocoaPods`
4. `tauri-cli` 与 iOS target

## 3. 验证清单

### 3.1 基础
- `git --version`
- `brew --version`
- `rustc --version`
- `cargo --version`

### 3.2 Apple / Tauri
- `xcodebuild -version`
- `pod --version`
- `cargo tauri --version`
- `rustup target list --installed | rg "aarch64-apple-ios|aarch64-apple-ios-sim"`

### 3.3 项目
- `cd /Users/codez/develop/yourConnector && cargo check --workspace`
- `cd /Users/codez/develop/yourConnector/app/mobile/src-tauri && cargo check`

## 4. 启动命令

1. `make run-relay`
2. `make run-sidecar`
3. `make run-mobile-tauri-ios IOS_SIM="iPhone 17 Pro"`

## 5. 协作规范

- 服务代码放在 `services/*`
- 协议代码放在 `protocol/rust`
- 协议 Schema 放在 `protocol/schema`
- 每次修改后先执行 `cargo check`
