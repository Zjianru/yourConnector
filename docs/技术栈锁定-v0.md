# yourConnector 技术栈锁定 v0

## 1. 锁定结论

### 1.1 App
- 仅保留一个 App：`Tauri Mobile + Rust`
- 首发平台：`iOS`
- 后续平台：`Android`
- UI 层：`HTML/CSS/TypeScript`（WebView）

### 1.2 服务层
- Relay：`Rust`（`services/relay`）
- Sidecar：`Rust`（`services/sidecar`）

### 1.3 协议层
- 协议代码：`protocol/rust`
- 协议 Schema：`protocol/schema`
- 原则：协议以 `protocol/rust` 为唯一代码源，Schema 与其对齐

## 2. 核心约束

- relay 不落库业务正文与终端输出
- 命令执行必须走白名单
- 终端必须支持可输入交互（PTY）

## 3. 版本策略

- Rust / Tauri 使用稳定版
- 依赖版本以 `Cargo.toml` / `Cargo.lock` 为准
- 每个里程碑结束后集中升级
